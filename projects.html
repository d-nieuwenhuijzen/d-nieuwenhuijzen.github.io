Brood Kaas en Ham

Brood kaas en Ham is een parodie spel van boter kaas en eieren wat ik als project heb gemaakt op Mondriaan. Ik heb geleerd hoe je automatisch een beurd doorgeeft en hoe je images kan gebruiken als knop. Hierbij heb ik vooral javascript gebruikt en HTML om de website optemaken.

Hoger Lager

Hoger Lager is een spel waar je een dobbelsteen moet gooien en een random getal krijgt en dan moet je raden of het getal wat je daarna gooit hoger of lager daarvan gaat zijn. Hierbij heb ik javascript, css en html gebruikt. Ik heb hier geleerd hoe je random getallen kan genereren.

MuseStat

MuseStat is een dashboard dat ik in een groepsproject heb gemaakt. Je kunt daar spotify stats zien van een paar artiesten. Ik heb hier geleerd hoe ik met MongoDB en localhost samen werk, ook heb ik Bootstrap 5 gebruikt en daardoor heb ik daar een stuk meer over geleerd. Ook heb ik ChartJS gebruikt en geleerd bij dit project.
